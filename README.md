# Engine9
The language includes built-in functions for common engineering formulas:
force(m, a) - computes force
kinetic_energy(m, v) - computes kinetic energy
ohm_current(v, r) - computes electrical current
work(f, d) - computes work
power(w, t) - computes power
pressure(f, a) - computes pressure
density(m, v) - computes density
momentum(m, v) - computes momentum

How to run:
Open the project folder in a terminal.
Run the Engine.py or type "python Engine.py" into terminal
When prompted type "sample"

This will execute the example program in sample.e9.
Modify sample.e9 file to continuing testing the language
To exit the program:
exit
